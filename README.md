# Atul Kumar Verma Portfolio

A responsive single-page developer portfolio for Atul Kumar Verma, focused on Laravel, MySQL, REST APIs, and full-stack web development.

## Overview

This project is a static portfolio website built with HTML, Tailwind CSS CDN, and a small amount of client-side JavaScript for interactions. It highlights professional summary, skills, journey, projects, resume access, and contact details.

## Features

- Responsive single-page layout
- Sticky header with mobile navigation drawer
- Hero section with profile summary and quick links
- Skills, journey, and project showcase sections
- Resume download support
- Contact section with email, phone, LinkedIn, and GitHub
- Smooth scroll and reveal animations

## Tech Stack

- HTML5
- Tailwind CSS via CDN
- Vanilla JavaScript
- Google Fonts

## Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/atulvermalal/portfolio.git
   ```

2. Open the project folder.
3. Run it with any static server, or open `index.html` directly in the browser.

Example using PHP local server:

```bash
php -S localhost:8000
```

Then open `http://localhost:8000`.
